# lcm-telop-turtlebot-demo

## Description:
Simple telop application for TurtleBot

## How to build
- mkdir build && cd build
- cmake ../
- make

## How to run
- ./build/bin/jacktelop-gui-demo (on client computer)
- ./build/bin/jacktelop-listener-demo (on TurtleBot computer)

## Dependences:
- LCM
- ROS
- CMake
- Qt4
